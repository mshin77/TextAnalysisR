# Semantic Document Clustering

Creates a unified visualization of document clustering with optional
clustering

## Usage

``` r
semantic_document_clustering(embeddings, method = "UMAP", clusters = NULL, ...)
```

## Arguments

- embeddings:

  Document embeddings matrix

- method:

  Dimensionality reduction method ("PCA", "t-SNE", "UMAP")

- clusters:

  Optional cluster assignments

- ...:

  Additional arguments
