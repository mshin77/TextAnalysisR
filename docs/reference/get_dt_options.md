# Default DT datatable options

Default DT datatable options

## Usage

``` r
get_dt_options(scroll_y = "400px", page_length = 25, show_buttons = TRUE)
```

## Arguments

- scroll_y:

  CSS height for vertical scrolling.

- page_length:

  Default page length.

- show_buttons:

  Include copy/csv/excel/pdf/print buttons.

## Value

A list for `DT::datatable(options = ...)`.
