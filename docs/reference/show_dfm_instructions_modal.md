# Show DFM instructions modal

Show DFM instructions modal

## Usage

``` r
show_dfm_instructions_modal(
  output_id,
  feature_name = "this feature",
  session = NULL
)
```

## Arguments

- output_id:

  Output id to reset after modal closes.

- feature_name:

  Feature description in the instructions.

- session:

  Shiny session.

## Value

Invisibly `NULL`.
