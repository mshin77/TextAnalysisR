# Truncate Text with Ellipsis

Truncates text to a maximum number of characters and adds ellipsis if
truncated.

## Usage

``` r
truncate_text_with_ellipsis(text, max_chars = 50)
```

## Arguments

- text:

  Character string to truncate.

- max_chars:

  Maximum number of characters (default: 50).

## Value

Truncated text with "..." appended if truncated.
