# Calculate Comprehensive Metrics

Calculates comprehensive similarity metrics including statistical
measures and network properties. Internal function used by
document_similarity_analysis.

## Usage

``` r
calculate_metrics(similarity_matrix, labels = NULL, method_info = NULL)
```

## Arguments

- similarity_matrix:

  A similarity matrix.

- labels:

  Optional vector of labels for clustering metrics.

- method_info:

  Optional method information.

## Value

A list of comprehensive metrics.
