# Single-cell message DT table

Single-cell message DT table

## Usage

``` r
create_message_table(message, font_size = "16px", color = "#6c757d")
```

## Arguments

- message:

  Character message to display.

- font_size:

  CSS font size.

- color:

  CSS color.

## Value

A [`DT::datatable`](https://rdrr.io/pkg/DT/man/datatable.html)
htmlwidget.
