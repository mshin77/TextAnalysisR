# An example structure of a structural topic model

Contains 15 topics, topic prevalences, and etc. from
[`stm::stm`](https://rdrr.io/pkg/stm/man/stm.html).

## Usage

``` r
stm_15
```

## Format

An object of class `STM` of length 11.

## Examples

``` r
stm_15
#> A topic model with 15 topics, 488 documents and a 4511 word dictionary.
```
