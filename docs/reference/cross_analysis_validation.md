# Cross Analysis Validation

Performs cross-validation on text analysis results

## Usage

``` r
cross_analysis_validation(results, verbose = FALSE, ...)
```

## Arguments

- results:

  Analysis results object to validate

- verbose:

  Logical indicating whether to print progress messages

- ...:

  Additional parameters

## Value

List containing validation status and metrics
