# Create Screen Reader Text

Generates visually hidden text for screen readers (WCAG 4.1.2).

## Usage

``` r
create_sr_text(text)
```

## Arguments

- text:

  Text to be read by screen readers

## Value

HTML span with sr-only class

## Examples

``` r
if (FALSE) { # \dontrun{
create_sr_text("Loading results, please wait")
} # }
```
