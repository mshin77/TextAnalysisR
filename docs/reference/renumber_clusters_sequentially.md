# Renumber Clusters Sequentially

Renumbers cluster assignments to sequential integers starting from 1.

## Usage

``` r
renumber_clusters_sequentially(clusters)
```

## Arguments

- clusters:

  A vector of cluster assignments.

## Value

Vector with clusters renumbered sequentially (1, 2, 3, ...).
