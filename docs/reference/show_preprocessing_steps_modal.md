# Show preprocessing-steps modal

Show preprocessing-steps modal

## Usage

``` r
show_preprocessing_steps_modal(
  title = "Preprocessing Required",
  message,
  required_steps,
  optional_steps = NULL,
  additional_note = NULL
)
```

## Arguments

- title:

  Modal title.

- message:

  Lead message.

- required_steps:

  Character vector of required step labels.

- optional_steps:

  Character vector of optional step labels.

- additional_note:

  Optional note appended to the body.

## Value

Invisibly `NULL`.
