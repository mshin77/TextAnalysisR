# Comprehensive Evaluation Metrics Calculator

Calculates comprehensive evaluation metrics for topic models including
neural coherence, LLM-based coherence, semantic diversity, and topic
stability measures.

## Usage

``` r
calculate_eval_metrics_internal(result, texts, selected_metrics)
```

## Arguments

- result:

  Topic modeling result object

- texts:

  Original text documents

- selected_metrics:

  Vector of metrics to calculate

## Value

List containing calculated evaluation metrics
