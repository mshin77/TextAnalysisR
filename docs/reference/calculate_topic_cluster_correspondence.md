# Calculate Topic-Cluster Correspondence

Computes Jaccard-based correspondence between topic keywords and cluster
keywords.

## Usage

``` r
calculate_topic_cluster_correspondence(topic_keywords, cluster_keywords, ...)
```

## Arguments

- topic_keywords:

  Topic keywords list.

- cluster_keywords:

  Cluster keywords list.

- ...:

  Additional parameters (currently unused).

## Value

List with correspondence metrics.
