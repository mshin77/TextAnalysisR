# Calculate Keyword Stability

Calculates stability between two sets of topic keywords.

## Usage

``` r
calculate_keyword_stability(keywords1, keywords2)
```

## Arguments

- keywords1:

  First set of keywords.

- keywords2:

  Second set of keywords.

## Value

Stability score.
